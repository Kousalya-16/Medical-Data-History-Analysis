# Medical Data History Analysis

## Project Overview
This project analyzes historical medical data to identify trends in patient demographics, diagnoses, and hospital admission patterns using SQL queries.

## Tools Used
- MySQL
- SQL

## Analysis Performed
- Analyzed patient demographics such as age and gender
- Examined diagnosis frequency and admission trends
- Applied joins, aggregations, and conditional logic to extract insights

## Key Insights
- Certain age groups show consistently higher admission rates
- Specific diagnoses recur frequently across patient segments
- Clear patterns observed in admission distribution

## Business Impact
The insights from this analysis can support better hospital resource planning, improve patient care strategies, and enhance operational efficiency.
